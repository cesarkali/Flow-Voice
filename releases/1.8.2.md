# FlowVoice v1.8.2

## Novidades e Melhorias

### 🎨 Melhorias de Design no Painel de Configurações
- **Links de Créditos do Desenvolvedor**: Adicionados links para o GitHub do projeto, o site pessoal do desenvolvedor e perfil do Instagram como créditos no rodapé da janela de configurações.
- **Redimensionamento Confortável**: Janela expandida de `620x640` para `650x750` para dar muito mais respiro visual às descrições dos campos e evitar cortes de textos.
- **Hierarquia Visual**: Todos os títulos das opções de configuração e caixas de seleção foram marcados em negrito (`font-weight: bold`), melhorando a legibilidade e contraste.
- **Reorganização de Dicas de API**: O bloco de ajuda de chaves de API foi atualizado com links clicáveis imediatos, classificando e recomendando o **Groq** e **GitHub Models** (como excelentes opções totalmente gratuitas) antes de Google Gemini e OpenAI.
- **Verificação de Atualização em Segundo Plano**: Adicionada verificação automática e assíncrona de novas atualizações a cada 1 hora, de forma extremamente leve e silenciosa.
- **Indicador de Status de Versão**: Exibição dinâmica da versão instalada no rodapé do painel de configurações (ex: `v1.8.2 (Atualizado)` ou sinalização de nova versão disponível), integrado ao sistema de busca assíncrono.

## Empacotamento

- **Windows:** `dist/FlowVoiceSetup.exe`
- **Ubuntu:** `ubuntu/dist/flowvoice_1.8.2_amd64.deb`

## Instalação

### Windows
Execute `FlowVoiceSetup.exe` ou aceite a atualização automática pelo popup no aplicativo.

### Ubuntu
```bash
sudo apt install ./ubuntu/dist/flowvoice_1.8.2_amd64.deb
```

> Comparar com a versão anterior: `releases/1.8.1.md`
