# FlowVoice v1.5.0

## Novidades

- Suporte ao **Ubuntu** com instalador `.deb` (`ubuntu/build-deb.sh`)
- Script único de build para Windows: `py build-windows.py`
- Versão centralizada em `version.py` para facilitar releases futuras

## Melhorias

- **Atualização automática no Windows:** ao aceitar a notificação, o app baixa e instala em modo silencioso, sem passos manuais no instalador
- **Desinstalação no Windows:** o instalador encerra o FlowVoice automaticamente antes de remover arquivos, evitando erro por app em execução
- Caminhos de configuração e logs multiplataforma (`%APPDATA%` no Windows, `~/.config/FlowVoice` no Ubuntu)
- Inicialização com o sistema no Linux via autostart `.desktop`

## Correções

- **Colar no primeiro Ctrl+V:** sincronização da área de transferência corrigida; o texto volta a ser colado imediatamente após a transcrição
- Ordem de colagem ajustada para não interferir com o overlay de status

## Empacotamento

- **Windows:** `dist/FlowVoiceSetup.exe`
- **Ubuntu:** `ubuntu/dist/flowvoice_1.5.0_amd64.deb`

## Instalação

### Windows
Execute `FlowVoiceSetup.exe` ou aceite a atualização automática quando notificado.

### Ubuntu
```bash
sudo apt install ./ubuntu/dist/flowvoice_1.5.0_amd64.deb
```

## Comparativo com v1.3.0

| Área | v1.3.0 | v1.5.0 |
|------|--------|--------|
| Plataformas | Windows | Windows + Ubuntu |
| Build | Comandos manuais (PyInstaller + Inno Setup) | Scripts automatizados |
| Atualização | Abria instalador interativo | Instalação silenciosa automática |
| Desinstalação | Podia falhar com app aberto | Encerra o app antes de desinstalar |
| Colar texto | Podia exigir segundo Ctrl+V | Cola no primeiro Ctrl+V |
