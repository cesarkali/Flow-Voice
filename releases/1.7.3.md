# FlowVoice v1.7.3

## Correções

- **Atualizador Automático:** Corrigido o bug onde o `FlowVoiceUpdater.exe` fechava subitamente na metade da atualização por conflito com o Inno Setup. O atualizador agora faz uma cópia temporária e se executa de forma segura e isolada.
- **Melhoria nos Logs:** Adicionado captura avançada de `sys.excepthook` e `traceback` inteiros.
- Correção no script do atualizador removendo duplicidade e conflito de inicialização do executável principal (`kill_app` dinâmico).

## Empacotamento

- **Windows:** `dist/FlowVoiceSetup.exe`
- **Ubuntu:** `ubuntu/dist/flowvoice_1.7.3_amd64.deb`

## Instalação

### Windows
Execute `FlowVoiceSetup.exe` ou aceite a atualização automática pelo popup no aplicativo.

### Ubuntu
```bash
sudo apt install ./ubuntu/dist/flowvoice_1.7.3_amd64.deb
```

> Comparar com a versão anterior: `releases/1.7.2.md`
