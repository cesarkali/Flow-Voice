# FlowVoice v1.8.0

## Novidades e Melhorias

### 🎨 Interface (FloatingOverlay)
- **Redesign completo do overlay de feedback**: O modal de confirmação (verde) e de erro (vermelho) que aparece após transcrição foi completamente reformulado com visual premium — gradientes, sombras suaves, ícones e tipografia modernos.
- **Exibição de texto sempre dentro do modal**: Corrigido o bug crítico onde o texto transcrito e as mensagens de erro ultrapassavam os limites do modal. O conteúdo agora fica sempre contido, com truncamento elegante (`…`) quando necessário.
- **Tamanho fixo e estável**: O modal agora tem dimensões consistentes independentemente do tamanho do texto, eliminando o comportamento instável de reposicionamento.
- **Mensagem de erro ao não detectar voz**: Adicionada mensagem de erro explícita e visível quando nenhuma fala é detectada na gravação, substituindo o silêncio anterior.

### 🎙️ Gravação e Transcrição
- **Indicador de gravação corrigido**: O indicador visual de gravação em andamento agora permanece centralizado e dentro do overlay, sem deslocar para fora do modal.

## Correções

- Corrigido overflow do texto transcrito para fora dos limites do modal de confirmação.
- Corrigido overflow da mensagem de erro para fora dos limites do modal de erro.
- Corrigida ausência de mensagem quando nenhuma voz é detectada.
- Corrigido comportamento instável de reposicionamento do overlay após a exibição inicial.

## Empacotamento

- **Windows:** `dist/FlowVoiceSetup.exe`
- **Ubuntu:** `ubuntu/dist/flowvoice_1.8.0_amd64.deb`

## Instalação

### Windows
Execute `FlowVoiceSetup.exe` ou aceite a atualização automática pelo popup no aplicativo.

### Ubuntu
```bash
sudo apt install ./ubuntu/dist/flowvoice_1.8.0_amd64.deb
```
