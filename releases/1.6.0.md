# FlowVoice v1.6.0

## Novidades

- Indicador visual durante instalação de atualização no Windows (janela com barra de progresso)
- Reinício automático do FlowVoice após concluir a atualização silenciosa

## Melhorias

- Overlay de status (gravando, polindo, etc.) mais estável — não some mais por timers antigos de sessões anteriores
- Texto ditado permanece na área de transferência após colar (Ctrl+C funciona com o conteúdo transcrito)

## Correções

- Pop-up de status que desaparecia durante gravação/processamento
- Colagem: removida restauração automática do clipboard antigo que apagava o texto ditado
- Sincronização reforçada da área de transferência antes e depois do Ctrl+V

## Empacotamento

- **Windows:** `dist/FlowVoiceSetup.exe`
- **Ubuntu:** `ubuntu/dist/flowvoice_1.6.0_amd64.deb`

## Instalação

### Windows
Execute `FlowVoiceSetup.exe` ou aceite a atualização automática quando notificado.

### Ubuntu
```bash
sudo apt install ./ubuntu/dist/flowvoice_1.6.0_amd64.deb
```

## Comparativo com v1.5.0

| Área | v1.5.0 | v1.6.0 |
|------|--------|--------|
| Atualização | Instalação silenciosa sem feedback | Janela "Instalando..." + reinício automático |
| Overlay | Podia sumir durante gravação | Permanece visível em gravando/polindo |
| Clipboard | Restaurava conteúdo antigo | Mantém texto ditado para Ctrl+C |
