# FlowVoice v1.7.2

## Novidades

- **`FlowVoiceUpdater.exe`**: executável dedicado para atualizações no Windows (substitui o script PowerShell)

## Melhorias

- Atualizador roda **separado** do app principal — não fecha junto com o FlowVoice
- Instalação com **permissão de administrador** (`RunAs`) para gravar em `Program Files`
- Aguarda o instalador terminar de fato antes de fechar a janela de progresso
- Reinicia o FlowVoice automaticamente após sucesso
- Log de diagnóstico em `%TEMP%\flowvoice_update.log`

## Correções

- Janela "Instalando..." que fechava antes da instalação concluir
- Atualização que falhava silenciosamente por falta de elevação
- App que não reabria após o update

## Empacotamento

- **Windows:** `dist/FlowVoiceSetup.exe` (inclui `FlowVoiceUpdater.exe` em `Program Files\FlowVoice\`)
- **Ubuntu:** `ubuntu/dist/flowvoice_1.7.2_amd64.deb`

## Instalação

### Windows
Execute `FlowVoiceSetup.exe` ou aceite a atualização automática quando notificado.

> **Importante:** se você está na v1.7.0 ou anterior **sem** `FlowVoiceUpdater.exe`, instale a v1.7.2 **manualmente uma vez**. Depois disso, as próximas atualizações serão automáticas.

### Ubuntu
```bash
sudo apt install ./ubuntu/dist/flowvoice_1.7.2_amd64.deb
```

## Comparativo com v1.7.0

| Área | v1.7.0 | v1.7.2 |
|------|--------|--------|
| Updater | Script PowerShell no temp | `FlowVoiceUpdater.exe` dedicado |
| Elevação admin | Não garantida | Sim (`RunAs`) |
| Reinício automático | Falhava | Corrigido |
| Log de erros | Não | `%TEMP%\flowvoice_update.log` |
