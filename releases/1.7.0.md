# FlowVoice v1.7.0

## Novidades

- Release de validação do **atualizador automático** no Windows

## Melhorias

- Versão incrementada para teste end-to-end do fluxo: notificação → download → instalação silenciosa → reinício do app

## Correções

- Nenhuma alteração funcional adicional nesta release (Ctrl+V e overlay estáveis desde a v1.6.0)

## Empacotamento

- **Windows:** `dist/FlowVoiceSetup.exe`
- **Ubuntu:** `ubuntu/dist/flowvoice_1.7.0_amd64.deb`

## Instalação

### Windows
Execute `FlowVoiceSetup.exe` ou aceite a atualização automática quando notificado.

### Ubuntu
```bash
sudo apt install ./ubuntu/dist/flowvoice_1.7.0_amd64.deb
```

## Comparativo com v1.6.0

| Área | v1.6.0 | v1.7.0 |
|------|--------|--------|
| Versão | 1.6.0 | 1.7.0 |
| Atualizador | Novo fluxo com indicador + reinício | Release para validar o fluxo completo |
| Colagem (Ctrl+V) | Corrigido | Mantido |

## Teste do atualizador (Windows)

1. Instale e abra a **v1.6.0** (ou versão anterior)
2. Publique a **v1.7.0** no GitHub Release com o asset `FlowVoiceSetup.exe`
3. No app, aceite a notificação de atualização
4. Confirme: janela "Instalando..." → instalação silenciosa → app reabre na v1.7.0
