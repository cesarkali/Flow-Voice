# FlowVoice v1.9.2

> Comparar com a versão anterior: [`releases/1.9.1.md`](1.9.1.md)

---

## Correções

- **Traduções quebradas no executável compilado (.exe):** No v1.9.1, o módulo `i18n.py` usava `os.path.dirname(sys.executable)` para localizar a pasta `locales/` quando rodando como `.exe`. O PyInstaller extrai os arquivos bundlados em `sys._MEIPASS`, que é um diretório temporário diferente da pasta do executável. Com isso, o `_load()` falhava silenciosamente, o cache de traduções ficava vazio e `tr(key)` devolvia a própria chave em vez do texto traduzido — resultando em toda a interface exibindo os identificadores brutos (`nav_connections`, `btn_save`, `field_api_key_label`, etc.). Corrigido usando `sys._MEIPASS` com fallback para `sys.executable`, igual ao padrão já adotado em `config.py`.

---

## Empacotamento

- **Windows:** `dist/FlowVoiceSetup.exe`

## Instalação

### Windows
Execute `FlowVoiceSetup.exe` ou use a atualização automática dentro do app.

### Ubuntu
```bash
sudo apt install ./ubuntu/dist/flowvoice_1.9.2_amd64.deb
```
